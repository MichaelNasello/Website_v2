Machine Learning keeps coming up, what is it?

Machine Learning is...

Machine learning is a subfield of Artificial Intelligence that involves the development of algorithms and data architectures that gives a computer program the power to 'learn' from data and make intelligent decisions without explicit programming. The training process involves the input of data to the model. For a specific instance of input, the model's output is compared to what the output should be. Using vector calculus, the internal parameters of the model are adjusted so that the model's output approaches the desired result. This process of training is repeated until the developer is pleased with the performance of their Machine Learning model.

The above description is simplified. For more detailed summaries of Machine Learning and Neural Networks, here are a few informative links: Neural Networks, Machine Learning for Beginners, 3blue1brown Video Tutorial.
